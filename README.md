# To-Do List Web Application

## Author
Leo Herrera

## Overview
This is a React-based web application that allows users to manage a to-do list. Users can add tasks, mark them as completed, delete tasks, and filter tasks by status. The app also includes a contact form built as a controlled component.

## Features
- Add new tasks
- Mark tasks as complete or incomplete
- Delete tasks
- Filter tasks (All / Active / Completed)
- Controlled contact form with multiple fields
- Navigation between pages using React Router

## Technologies Used
- React (Create React App)
- JavaScript
- HTML
- CSS

## How to Use
1. Navigate to the Todos page
2. Enter a task and click "Add Task"
3. Use buttons to complete or delete tasks
4. Use filters to view different task types
5. Visit the Contact page to fill out the form
